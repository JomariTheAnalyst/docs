## How to access the admin

The Shopify Admin is where you manage everything — products, orders, collections, apps, and settings.

{/* Component: CodeGroup */}
<CodeGroup>

```text Admin URL
https://sixthgearmoto.myshopify.com/admin
```

```text Login Page
https://accounts.shopify.com/store/sixthgearmoto/login
```

</CodeGroup>

{/* Component: Note */}
<Note>
You need a Shopify user account to sign in. Contact the store owner if you do not have access. User access is managed under **Settings → Users**.
</Note>

## Headless setup

{/* Component: Info */}
<Info>
- The live storefront is a Next.js application that reads commerce data from Shopify through the Storefront API.
- The Horizon theme is used for Shopify preview and fallback only. It is not the live SixthGear website.
- Product, collection, inventory, and supported metafield changes become available to the live website after the Next.js application fetches updated Shopify data.
- Edit commerce data in Shopify Admin. Do not edit Horizon theme files expecting them to update the live headless storefront.
</Info>

{/* Component: CodeGroup */}
<CodeGroup>

```text Storefront API Endpoint
https://sixthgearmoto.myshopify.com/api/2025-10/graphql.json
```

</CodeGroup>
