Sanity image fields can include crop, hotspot, alt text, and separate mobile media. The exact fields vary by content type.

## Prepare an image

- Use an approved source with documented usage rights.
- Remove customer, order, credential, and private campaign information.
- Export at a suitable size before upload; do not upscale a small source.
- Keep text out of the image when editable text can appear in the page.

## Update media

<Steps>
  <Step title="Open the target document">
    Confirm the image belongs to the document and section you intend to change.
  </Step>
  <Step title="Upload or select the asset">
    Choose the approved image. Avoid duplicates when the same asset already exists.
  </Step>
  <Step title="Set crop and hotspot">
    Keep the subject visible across desktop and mobile crops. Use a dedicated mobile image when the schema provides one and the composition requires it.
  </Step>
  <Step title="Write alt text">
    Describe the useful visual information in a short sentence. Do not repeat nearby headings or use a file name as alt text.
  </Step>
  <Step title="Publish and verify">
    Check image quality, crop, loading, and layout on representative desktop and mobile widths.
  </Step>
</Steps>

<Warning>
Do not upload screenshots that contain account menus, tokens, customer details, orders, billing information, or unpublished confidential campaigns.
</Warning>
